Basic Chatbot

This project is a Basic Chatbot that can interact with users using
simple predefined responses.

Features:
- Responds to basic user inputs
- Uses simple conditions (if-else)
- Easy to understand for beginners

How it works:
- User enters a message
- The chatbot checks the input
- A predefined response is returned

How to run:
1. Install the required software (Python / Java)
2. Open the project folder
3. Run the main file

Example (Python):
python chatbot.py

Use case:
This project is useful for beginners to learn basic programming logic
and understand how chatbots work.
